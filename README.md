# siteBiblioteca

## Prompt Atualizado

### Responsividade (Obrigatório)
A página deve ser **100% responsiva**, adaptando-se perfeitamente a qualquer tamanho de tela, desde smartphones até monitores ultrawide.

Implemente um layout **mobile-first**, utilizando CSS moderno (Flexbox e CSS Grid), garantindo uma ótima experiência em:

- Smartphones (320px em diante)
- Tablets
- Notebooks
- Monitores Full HD e Ultrawide

Requisitos obrigatórios:

- O menu deve se transformar em um **menu hambúrguer** em telas menores.
- Os cards dos livros devem se reorganizar automaticamente conforme o tamanho da tela.
- Imagens devem ser responsivas, sem distorção ou perda de qualidade.
- Botões e links devem possuir tamanho adequado para toque em dispositivos móveis.
- Textos devem ajustar automaticamente o tamanho para manter a legibilidade.
- Não deve existir rolagem horizontal em nenhuma resolução.
- Todos os elementos devem permanecer alinhados e proporcionais em qualquer dispositivo.
- O espaçamento (padding e margin) deve se adaptar conforme o tamanho da tela.
- O banner principal (Hero) deve reorganizar seus elementos para manter uma boa aparência em telas pequenas.
- O rodapé também deve se adaptar, reorganizando as colunas quando necessário.

Utilize **media queries** para os principais pontos de quebra:

- até 480px
- até 768px
- até 1024px
- acima de 1024px

Faça com que o site tenha aparência e funcionamento de um site profissional moderno, semelhante a grandes e-commerces, proporcionando excelente experiência tanto em dispositivos móveis quanto em computadores.
